# 1. Cybersecurity Fundamentals

## CIA Triad

### Confidentiality
Protecting sensitive information from unauthorized access.

### Integrity
Ensuring data remains accurate and unmodified.

### Availability
Ensuring systems and data are accessible when needed.

---

# 2. Common Threat Types

| Threat Type | Description |
|-------------|-------------|
| Phishing | Fake emails/websites used to steal credentials |
| Malware | Malicious software designed to damage systems |
| DDoS | Flooding a server/network with traffic |
| SQL Injection | Injecting malicious SQL queries into applications |
| Brute Force | Repeated password guessing attacks |
| Ransomware | Encrypts files and demands payment |

---

# 3. Attack Vectors

## Social Engineering
Manipulating users into revealing confidential information.

## Wireless Attacks
Attacks on Wi-Fi networks such as packet sniffing and rogue access points.

## Insider Threats
Threats originating from employees or trusted users.
